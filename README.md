# Cloud Arcade ☁️🎮

A lightweight, unblocked web games portal built with HTML, CSS, and JavaScript. 

## 🚀 Live Link
[Click here to play!](https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/)

## 🛠️ Features
- **Stealth Mode**: Tab name and icon automatically change to "Google Drive" to blend in.
- **Panic Key**: Press the **ESC** key or the red **PANIC** button to instantly redirect to Google Classroom.
- **Cloak Mode**: Open games in an `about:blank` tab to hide them from browser history.
- **Searchable Grid**: Easily find games by typing in the search bar.

## 🕹️ Controls
| Action | Key / Button |
| :--- | :--- |
| **Panic Redirect** | `Escape` (ESC) |
| **Open Game** | Click Game Card |
| **Stealth Launch** | Click "Cloak Open" |
